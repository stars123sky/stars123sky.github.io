```markdown
# Markdown 语法总结
## 标题
Markdown 支持六级标题，分别对应 HTML 中的 `<h1>` 到 `<h6>` 标签。在文字前面加上 `#` 即可，`#` 的数量代表标题的级数。
```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```
## 段落与换行
直接编写文字，换行时输入两个空格然后回车，或者直接另起一行。
```markdown
这是一个段落。
这是另一个段落。
```
## 强调
星号 `*` 或下划线 `_` 包裹文本表示强调，对应 HTML 的 `<em>` 标签。
```markdown
*斜体* 或 _斜体_
**粗体** 或 __粗体__
```
## 列表
- 无序列表使用星号 `*`、加号 `+` 或减号 `-` 作为列表标记：
```markdown
* 列表项一
* 列表项二
* 列表项三
```
- 有序列表使用数字后面跟上英文句点和空格：
```markdown
1. 第一项
2. 第二项
3. 第三项
```
## 链接与图片
- 链接 `[显示文本](链接地址 "标题")`：
```markdown
[百度](http://www.baidu.com "百度一下")
```
- 图片 `![替代文字](图片链接 "标题")`：
```markdown
![这是一张图片](http://example.com/image.jpg "示例图片")
```
## 引用
在引用的文字前加上 `>` 符号，并且可以有多个 `>` 表示层级更深：
```markdown
> 这是一个引用。
>
>> 这是一个嵌套的引用。
```
## 代码
- 行内代码使用一对反引号 `` ` `` 包裹：
```markdown
这是行内代码 `code`。
```
- 代码块使用三个反引号 ```` ``` ```` 包裹，并指定语言（可选）：
```markdown
```python
print("Hello, Markdown!")
```
```
## 表格
使用竖线 `|` 和短横线 `-` 创建表格，表头可以使用 `-` 和 `:` 设置对齐方式：
```markdown
| 左对齐 | 居中对齐 | 右对齐 |
| :----- | :------: | -----: |
| 单元格 | 单元格   | 单元格 |
| 单元格 | 单元格   | 单元格 |
```
## 分隔线
使用三个以上的星号 `*`、减号 `-` 或下划线 `_` 表示分隔线：
```markdown
---
---
```
## HTML代码
Markdown 支持直接插入 HTML 代码，但某些平台可能会限制这一功能：
```markdown
<p>这是一个HTML段落。</p>
```
## 转义字符
使用反斜杠 `\` 转义特殊字符：
```markdown
\*这不是强调\*
```
这些是Markdown的基础语法，适用于大多数Markdown编辑器和平台。
```
这个总结包含了Markdown的基本语法，可以直接在支持Markdown的编辑器中使用。
